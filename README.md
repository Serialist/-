# STM32的简单麦克纳姆轮小车

## 简介

这个是一个由 STM32 最小单片机实现的简单麦克纳姆轮小车，包括了简单的 PWM 波输出和麦轮解算操作。

### 实验材料

- STM32F103C8T6 单片机最小系统板
- L298N 电机驱动模块
- LM2596 降压模块
- GBJ37-520 电机
- PS2 手柄及其接收器
- 小车底盘
- Keil 5
